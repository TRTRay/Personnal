
## 登录

**接口地址**: /api/users/tokens

**请求方式**: POST

**consumes**: ["application/json"]

**produces**: ["\*/\*"]

**请求参数**：

|参数名称 |参数说明   |是否必须 |数据类型 |
| :------- | :--------- | -------- | -------- |
| userPhone |用户手机号 |true     |string   |
| userPassword |用户密码   | true | string |

 **响应示例**:

``` JSON
{
    "status": 200,
    "message":"success!",
    "success":true,
    "data":{
        "token":"string"
    }
}
```

**data响应参数**:

| 参数名称 | 参数说明      | 类型   |
| -------- | ------------- | ------ |
| token    | 返回用户token | string |
