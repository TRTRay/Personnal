## 获取当日打卡记录

**接口地址**: /api/signinfo

**请求方式**: GET

**consumes**: ["application/json"]

**produces**: ["\*/\*"]

**请求参数**：

| 参数名称  | 参数说明             | 是否必须 | 数据类型 |
| :-------- | :------------------- | -------- | -------- |
| userPhone | 用户手机号           | true     | string   |
| date      | 打卡日期(YYYY-MM-DD) | true     | string   |

**响应示例**:

``` JSON
{
    "status": 200,
    "message":"success!",
    "success":true,
    "data":{
        "signed" : true,
        "step" : 100
    }
}
```

**响应参数**:

| 参数名称 | 参数说明             | 数据类型 |
| :------- | :------------------- | -------- |
| signed   | 用户当日是否已经打卡 | boolean  |
| step     | 用户打卡时运动步数   | integer  |
